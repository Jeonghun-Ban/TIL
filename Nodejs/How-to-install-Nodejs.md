How to install Nodejs
===

Install Nodejs
---

1. Launch Terminal
2. Install Nodejs using apt package manager

    ```bash
    sudo apt install nodejs
    ```
3. Verify the installation

    ```bash
    nodejs --version
    ```

Install npm
---

1. Install npm using apt package manager

    ```bash
    sudo apt install npm
    ```
2. Verify the installation

    ```bash
    npm --version
    ```
